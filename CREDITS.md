# Créditos y reconocimientos

Merlos CAD es posible gracias a las siguientes tecnologías de código abierto:

| Tecnología | Licencia | Uso en el proyecto |
|---|---|---|
| [Python](https://www.python.org/) | PSF | Lenguaje base |
| [Tkinter](https://docs.python.org/3/library/tkinter.html) | PSF | Interfaz gráfica |
| [Pillow](https://python-pillow.org/) | HPND | Renderizado de imágenes |
| [PyOpenGL](https://pyopengl.sourceforge.net/) | BSD | Renderer OpenGL |
| [NumPy](https://numpy.org/) | BSD | Cálculo numérico |
| [ezdxf](https://ezdxf.readthedocs.io/) | MIT | Importar y exportar archivos DXF |
| [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter) | MIT | Componentes de interfaz modernos |
| [pywin32](https://github.com/mhammond/pywin32) | PSF | Integración con Windows |
